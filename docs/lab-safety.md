# 安全实验边界

本计划会涉及敏感文件读取、网络外联、危险命令、资源滥用、删除和重命名文件等实验。所有实验必须可控、可解释、可复现。

## 允许范围

- 敏感文件实验使用 `lab/secrets/` 下的模拟文件。
- 文件读写、删除、重命名实验只作用于 `lab/workspace/`。
- 网络外联只访问本地测试服务或明确允许的测试地址。
- 资源滥用样例必须有 timeout、数量限制或输出限制。
- 公开 demo payload 必须脱敏，并能在本地安全复现。

## 禁止范围

- 不读取真实 SSH key、云凭证、浏览器 cookie、真实业务配置。
- 不执行不可控的破坏性命令。
- 不对公网目标做扫描、爆破、利用或未授权请求。
- 不把真实密钥、token、个人数据写入样例或日志。

## 破坏性样例处理

危险命令优先做字符串检测或受控样例。确需执行时，必须满足：

- 作用目录限定在 `lab/workspace/`
- 有 timeout 或资源限制
- 能从 README 或实验记录中看清楚影响范围
